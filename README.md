Project Description: FocusLock
1. Specific Application Area
The interactive mobile application to be developed falls within the Productivity and Digital Wellbeing sector, focusing specifically on improving study and work habits. The system functions as a time management and distraction-blocking tool, utilizing the smartphone's built-in sensors to encourage the user to leave the device inactive during periods of "Deep Work" or Pomodoro sessions.
The application leverages the physical restriction of smartphone usage as a catalyst for concentration, promoting self-discipline through positive and negative audio feedback.

2. Application Objectives
Primary Objective: To help the user maximize concentration time and build sustainable focus habits.
Specific Objectives:

Maximize Concentration Time: Encourage uninterrupted study/work by forcing the user to physically step away from the screen.

Discourage Compulsive Checking: Apply negative audio feedback (e.g., a short alarm) if the user picks up the phone before the focus time is over.

Create a Conducive Sound Environment: Play Lo-Fi music, white noise, or nature sounds while the device remains in the correct focus position.

Increase Performance Awareness: Present visual performance statistics (total focus time, recorded breaks, and weekly progress), reinforcing progress and self-evaluation.

3. Features to Develop
To achieve the defined objectives, the application will combine mobile sensor reading with adaptive audiovisual feedback:

Focus Position Detection — Proximity Sensor: The focus mode is only activated when the screen is facing down, detected by the obstruction of the front sensor.

Interruption Monitoring — Accelerometer: The system identifies sudden movements during the session, interpreting them as breaks in focus.

Reactive Multimedia Feedback:

During Focus: Continuous playback of relaxing audio; dark and serene graphical environment.

Upon Interruption: Immediate halt of the music and playback of an alert sound.

Visual Statistics and Focus History:

Total daily/weekly focus time.

Number of recorded breaks.

Percentage of sessions completed without interruption.

Evolution charts (bar charts or progress rings).
(Objective: To reinforce motivation and track personal progress).
